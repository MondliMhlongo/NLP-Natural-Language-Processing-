# NLP-Natural-Language-Processing-

Sentiment Analysis (Task 9)
Overview

This project was completed as part of the Codveda Technologies Data Analytics Internship Program. It demonstrates the use of Natural Language Processing (NLP) techniques to analyse customer reviews and classify their sentiment as positive, negative, or neutral.

Objectives
Perform sentiment analysis on textual data.
Classify customer reviews based on sentiment.
Generate sentiment scores using Natural Language Processing.
Visualise the distribution of customer sentiments.
Tools & Technologies
Python
Pandas
NLTK (VADER Sentiment Analyzer)
Matplotlib
Tasks Performed
Imported and prepared the customer review dataset.
Applied the VADER Sentiment Analyzer to calculate compound sentiment scores.
Classified each review as Positive, Negative, or Neutral.
Stored sentiment scores and classifications in the dataset.
Analysed the overall sentiment distribution.
Visualised the results using a bar chart.
Learning Outcomes

Through this project, I gained practical experience in:

Natural Language Processing (NLP).
Sentiment analysis using VADER.
Text data preprocessing.
Feature extraction from textual data.
Visualising sentiment trends for data-driven insights.
